Univariate Bernstein regression
===============================

This example visualizes how Bernstein features approximate a univariate function while reporting the mean
absolute error on a dense evaluation grid.

Run it from the project root with:

.. code-block:: bash

   uv run python examples/univariate_bernstein_regression.py
