Legendre classification
=======================

This script compares Legendre polynomial features with logistic regression on the classic two-moons dataset.
It enables the tensor-product expansion to capture interactions between the two input coordinates.

Run it from the project root with:

.. code-block:: bash

   uv run python examples/legendre_classification.py
